# Drixel Studio - Ultra-Modern Design Showcase

A premium, futuristic portfolio website for UI/UX Designer & Full Stack Developer featuring:

## ✨ Features

- **Dark Premium Theme** with electric neon gradients
- **Glassmorphism** cards with subtle blur effects
- **Smooth Scroll** powered by Lenis
- **Advanced Animations** using Framer Motion & GSAP
- **Custom Cursor** with magnetic interactions
- **Dribbble-Style Grid** with masonry layout
- **Horizontal Scroll** branding showcase
- **Interactive Modals** for case studies
- **60fps Performance** optimized animations
- **Fully Responsive** with mobile optimizations

## 🎨 Color System

- Deep Space Black: `#0B0F1A`

- Electric Purple: `#7C3AED`
- Neon Blue: `#06B6D4`
- Hot Pink: `#EC4899`
- Soft Cyan: `#22D3EE`

## 🚀 Tech Stack

- React 18 (Vite)
- Tailwind CSS
- Framer Motion
- GSAP + ScrollTrigger
- Lenis (Smooth Scroll)
- Lottie React

## 📦 Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
